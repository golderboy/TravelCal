โครงสร้างไฟล์
- index.php                หน้าแสดงผลหลัก
- data/security.php        กฎความปลอดภัยและ security headers
- assets/css/app.css       สไตล์แยกจากหน้าแสดงผล
- assets/js/app.js         JavaScript คำนวณแยกจากหน้าแสดงผล

วิธีใช้งาน
1) วางโฟลเดอร์ travel_calculator ไว้ใต้ web root
2) เปิดผ่าน HTTPS จะเหมาะที่สุด เพราะ security.php ตั้ง session.cookie_secure = 1
3) ถ้าทดสอบบน HTTP ชั่วคราว ให้ปรับ session.cookie_secure เป็น 0 ใน data/security.php

สิ่งที่ระบบทำ
- คำนวณระยะเวลาเดินทางจากวันเวลาออกเดินทางถึงวันเวลาถึงที่พัก
- คำนวณค่าเดินทางรถยนต์ 4 บาท/กม. และรถจักรยานยนต์ 2 บาท/กม.
- รับระยะทางขาเดียวและคูณ 2 เป็นไป-กลับ
- ปัดเศษลงเป็นบาทถ้วนหลังรวมค่าเดินทางแล้ว
